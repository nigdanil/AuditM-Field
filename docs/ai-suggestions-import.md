# AI suggestions import

AuditM-Field does not call AI providers directly from the PWA.

Recommended flow:

```text
AuditM-Field
  -> ZIP export
  -> n8n/backend
  -> OCR/CV/LLM/RAG processing
  -> ai-suggestions.json
  -> AuditM-Field import
  -> human review in Annotator
```

## Quick local test

Open:

```text
Export Center -> AI suggestions import
```

Click:

```text
Generate demo suggestions
```

AuditM-Field will:

```text
- take the latest local photo from IndexedDB;
- find its inspection;
- generate ai-suggestions_<inspectionId>_<photoId>.json;
- download the file.
```

Then upload the downloaded file using:

```text
Import AI suggestions
```

Expected result:

```text
Imported 1/1 AI suggestions.
Open first affected photo.
```

Open the affected photo in Annotator and check that the imported annotation has:

```text
source: ai
```

## Minimal JSON format

```json
{
  "schemaVersion": "1.0.0",
  "source": "ai",
  "provider": "n8n-demo",
  "inspectionId": "inspection-id",
  "createdAt": "2026-05-10T00:00:00.000Z",
  "suggestions": [
    {
      "id": "ai-suggestion-001",
      "photoId": "photo-id",
      "type": "product_area",
      "label": "Product area",
      "shape": "rectangle",
      "confidence": 0.82,
      "rawAnnotation": {
        "id": "ai-suggestion-001",
        "type": "Annotation",
        "target": {
          "selector": {
            "type": "FragmentSelector",
            "conformsTo": "http://www.w3.org/TR/media-frags/",
            "value": "xywh=pixel:120,80,300,220"
          }
        }
      },
      "attributes": {
        "brand": "Our brand",
        "condition": "OK",
        "comment": "Generated by demo AI pipeline"
      }
    }
  ],
  "report": {
    "summary": "Demo AI pipeline detected one product area.",
    "warnings": []
  }
}
```

## Import behavior

AuditM-Field validates:

```text
- JSON shape;
- source === "ai";
- inspectionId exists in local IndexedDB;
- every suggestion.photoId exists in that inspection.
```

Then it saves suggestions as annotations:

```text
source: "ai"
```

The user can open Annotator and review/edit/delete them.
