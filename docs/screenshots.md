## Screenshot style

Recommended browser width:

```text
1440px or wider
```

Recommended capture areas:

```text
- full page for documentation
- focused component for README
```

Avoid screenshots with:

```text
- personal data
- real customer data
- real store names
- real brand names if not allowed
```
